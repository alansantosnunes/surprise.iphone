# 💍 Surpresa App — "Quer casar comigo, Gatinha?"

App React Native gerado a partir do site original. Roda no iPhone via Expo Go (sem precisar de Mac ou Xcode!).

---

## 🚀 Como rodar no iPhone

### 1. Instale o Node.js
Baixe em: https://nodejs.org (versão LTS)

### 2. Instale o Expo Go no iPhone
Busque **"Expo Go"** na App Store e instale.

### 3. Instale as dependências
Abra o terminal na pasta do projeto e rode:
```bash
npm install
```

### 4. Inicie o app
```bash
npx expo start
```

### 5. Abra no iPhone
- Um QR Code vai aparecer no terminal
- Abra a câmera do iPhone e aponte para o QR Code
- O app abre automaticamente no Expo Go 🎉

---

## ✨ O que o app faz
- Pergunta "Quer casar comigo, Gatinha? *-*"
- Botão **Sim**: mostra animação e mensagem de amor ❤️
- Botão **Não**: foge do dedo quando você tenta apertar 😂

---

## 📦 Publicar na App Store (opcional)
Para publicar de verdade na App Store, você vai precisar de:
- Mac com Xcode instalado
- Conta de desenvolvedor Apple (U$ 99/ano)
- Rodar: `eas build --platform ios`

Mas para a surpresa, o Expo Go já resolve perfeitamente! 💕
